# AGV / AMR Wheel Drive Gearbox

SigGear provides precision gearboxes and compact drive solutions for AGV, AMR, mobile robots, wheel drive modules, and industrial automation systems.

AGV and AMR wheel drive systems require reliable torque output, compact size, stable motion, long service life, and easy integration with motors, wheels, encoders, brakes, and control systems. SigGear supports robotic and automation customers with precision reducers, CAD models, product documentation, and engineering resources for fast evaluation and prototype development.

## Why AGV and AMR Wheel Drives Need Reliable Gearboxes

AGV and AMR robots are widely used in warehouses, factories, logistics centers, hospitals, laboratories, and automated production lines. The wheel drive gearbox plays an important role in converting motor speed into usable wheel torque while supporting smooth movement, load carrying, acceleration, braking, and repeated operation.

A suitable gearbox helps improve:

* Wheel torque output
* Driving stability
* Low-speed control
* Load carrying capability
* Service life
* Compact drive module design
* Motor matching
* Prototype development speed
* Overall mobile robot reliability

For AGV and AMR applications, the gearbox should be selected according to payload, wheel diameter, speed, torque, duty cycle, installation space, motor type, brake requirement, and operating environment.

## Selection Parameters

When selecting a gearbox for AGV and AMR wheel drive systems, engineers should evaluate both mechanical and operating requirements.

| Parameter            | Notes                                                                             |
| -------------------- | --------------------------------------------------------------------------------- |
| Robot payload        | Check the maximum load carried by the AGV or AMR.                                 |
| Wheel diameter       | Affects required torque, speed, and drive module layout.                          |
| Output torque        | Match starting torque, continuous torque, and peak torque requirements.           |
| Output speed         | Confirm target robot speed and wheel speed.                                       |
| Gear ratio           | Balance torque, speed, efficiency, and motor operating range.                     |
| Duty cycle           | Evaluate continuous operation, start-stop frequency, and working hours.           |
| Slope requirement    | Consider ramp climbing, floor conditions, and load changes.                       |
| Noise requirement    | Important for indoor, hospital, laboratory, and commercial environments.          |
| Installation space   | Check drive module diameter, axial length, and mounting space.                    |
| Motor compatibility  | Confirm motor size, shaft design, encoder, brake, and integration method.         |
| Wheel interface      | Confirm output shaft, flange, hub, and wheel mounting design.                     |
| Lifetime requirement | Consider operating environment, daily working time, and maintenance expectations. |

## AGV / AMR Drive System Considerations

AGV and AMR drive systems may use different structures depending on payload, cost, speed, and space requirements.

| Drive Design                     | Typical Consideration                                                        |
| -------------------------------- | ---------------------------------------------------------------------------- |
| Integrated wheel drive module    | Compact design, faster assembly, and easier replacement.                     |
| Motor + gearbox + wheel assembly | Flexible design for custom mobile robot platforms.                           |
| Dual-wheel differential drive    | Common for AGV and AMR platforms requiring simple steering.                  |
| Steering wheel drive module      | Useful for omnidirectional or advanced mobile robot movement.                |
| Low-noise indoor drive           | Important for hospitals, offices, laboratories, and commercial environments. |
| Heavy-load AGV drive             | Requires higher torque, stronger bearings, and reliable long-term operation. |
